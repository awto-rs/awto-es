Language Agnostic

Thalo runs wasm components, which means if your language supports wasm, then so does Thalo.

Performant

Written in Rust, Thalo is built for performance through caching and actors.

Postgres Message Store

Thalo uses Message DB as the message store, which is built on the well-known & battle tested database, Postgres.

What is Thalo?

Thalo is a runtime that uses WASM to provide a language-agnostic framework for building event-sourced applications. It simplifies event sourcing by abstracting away unnecessary complications and providing a simple, yet powerful API for developers to use.

What is Event Sourcing?

Event sourcing is a software design pattern in which the state of an application is derived from a log of events. In this pattern, events are recorded in an append-only store and can be used to reconstruct the current state of the application or to replay past events to understand the history of the application.
